**Definition:**

### 2. The Three Basic Structures
### 'D*1'CJ( 'D#3'3J) 'D+D'+)

#### A. Sequence ('D*3D3D)
Execute steps one after another in order.

**Example:** Calculate average
```
Input sum
average = sum / 6
Output average
```

#### B. Decision ('DB1'1)
Choose between alternatives based on a condition.

**Example:** Even or Odd
```
IF number mod 2 = 0
    Print "Even"
ELSE
    Print "Odd"
```

#### C. Repetition ('D*C1'1)
Repeat steps while condition is true.

**Example:** Sum of scores
```
i = 1
WHILE i <= 6
    Input score
    Add to sum
    i = i + 1
ENDWHILE
```
